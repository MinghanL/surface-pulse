# Surface Pulse

**[Live Demo →](https://minghanl.github.io/surface-pulse/)**

A touch-based material detection interface that communicates with a sensor device over Bluetooth Low Energy. Place material stickers on the canvas, touch them with the sensor, and watch the readings come in live.

---

## Features

- **BLE Connection** — pair with a sensor device directly from the browser via Web Bluetooth
- **Material Canvas** — drag glass, wood, metal, rubber, fabric, stone, or sand stickers onto a freeform canvas
- **Live Data Monitor** — floating panel showing real-time material type, touch area, and force with a waveform chart
- **Touch Detection** — visual ring indicator highlights which sticker is being touched
- **Templates** — one-tap presets to load common material layouts
