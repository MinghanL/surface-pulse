A **touch-based material detection interface** that wirelessly communicates with sensor hardware via **Web Bluetooth**.  
Drag material stickers onto the canvas, interact with a physical touch sensor, and view real-time feedback in the browser.

---

## ✨ Features

- 🎯 **BLE Connectivity** — Pair directly with your sensor device from any modern browser
- 🎨 **Interactive Canvas** — Drag & drop material stickers (glass, wood, metal, rubber, fabric, stone, sand)
- 📊 **Live Data Monitor** — Floating panel with real-time touch area, material type, and waveform visualization
- 🔘 **Touch Indicator** — Animated ring highlights active material detection
- 🧩 **Template Presets** — One-click setup for common material layouts

---

## 🚀 How It Works
1. Open the [Live Demo](https://minghanl.github.io/surface-pulse/) in a supported browser  
2. Connect your BLE sensor device  
3. Add material stickers to the canvas  
4. Touch the physical sensor to send real-time data to the interface
